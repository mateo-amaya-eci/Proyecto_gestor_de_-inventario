Descripción detallada

Gestiona la conexión y operaciones con la base de datos SQLite.

La clase DatabaseManager proporciona métodos para interactuar con la base de datos, incluyendo operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para los componentes.
